My UX Skills

User Research

Based on the scope of the project and information needed, I normally utilize 2-7 user interviews. I also conduct google surveys in order to collect quantitative data

User Empathy

To empathzie with my users pain points. I create an affinity diagram to organize thoughts and recognize patterns. I then create an empathy map and user journey map to visualize their struggles.

Qualitative Interviews

I create an interview plan with a general script of what questions I want to ask. I ask users permission to record the interviews so I can reference the information provided.
